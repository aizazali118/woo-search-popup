Installation

Up in 3 steps

Step 01
📦
Upload Plugin
WordPress Admin → Plugins → Add New → Upload ZIP → Activate
Step 02
📋
Copy Shortcode
Settings → Search Popup → Copy any shortcode with one click
Step 03
✅
Paste & Done

Shortcodes

[wsp_search_button label="Search"] Icon + label
[wsp_search_button label="Find Products"] Custom label
